### Isolation verified in-run
The second run added `assert_pristine` (in `run.sh`), which asserts each repo matches its snapshot before every cell. All 8 checks (2 repos × 4 arms) printed `pristine ✓` — live proof that no arm's edits leak into the next. Combined with fresh `claude -p` per cell (no `--continue`/`--resume`) and `--setting-sources project,local` (claude-mem / user memory excluded — confirmed by grepping the transcripts: no memory tooling, no hooks), the arms are fully independent.
